Feelit

  • A web interface for patterns looking-up in LJ40K

  • make sure the current internet is under the *.iis.sinia.edu.tw domain

Usage

# use default setting
# port: 5000
# debug: false
python server.py

# enter debug mode
python server -d # or --debug

# specified port number
python server -p 8888
